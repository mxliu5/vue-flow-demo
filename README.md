
## 命令

- npm i 安装依赖包
- npm run dev 本地执行
- npm run build 打包构建
- npm run docs:dev 本地打开框架使用说明文档
- npm run docs:build 打包构建框架使用说明文档
- npm run docs:preview 预览框架使用说明文档
